## 7. Key Features Summary

### âœ… What This Workflow Does:

1. **Creates VM** from image with Qualys agent pre-installed
2. **Validates** the image using scripts from Git repository
3. **Removes Qualys Agent** if validation passes:
   - Stops and disables service
   - Removes all agent files
   - Cleans up systemd configuration
4. **Creates New Image** without Qualys agent
5. **Promotes Image** to display project with `ephemeral="true"` label
6. **Cleans Up** validation VM

### ğŸ¯ Key Differences from Other Workflows:

- **Source Image**: Uses `test-rhel-vm-agent-image` (with Qualys agent)
- **Agent Removal**: Explicitly removes Qualys agent after validation
- **New Image Creation**: Creates `{source}-no-agent` image
- **Ephemeral Label**: Promoted image has `ephemeral="true"` label
- **Conditional Flow**: Only removes agent and promotes if validation passes
